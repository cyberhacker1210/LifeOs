# Sound Assets

Please add the following audio files to this directory (`public/sounds/`):

1. `pop.mp3` - A short, satisfying pop sound for completing tasks.
2. `alarm.mp3` - A ringtone or chime for when the timer finishes.

You can download free sounds from [freesound.org](https://freesound.org/) or [mixkit.co](https://mixkit.co/free-sound-effects/).
